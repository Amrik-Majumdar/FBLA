## üì± Deployment Commands

### Development
```bash
# Install dependencies
npm install

# Start development server
npm start

# Start Expo
expo start
```

### Build for Production
```bash
# Build for all platforms
expo build

# Build for specific platform
expo build:android
expo build:ios
expo build:web

# Build with preview
expo build --platform all --profile preview
```

### Deployment with EAS
```bash
# Install EAS CLI
npm install -g eas-cli

# Build and deploy
eas build --platform all

# Deploy to stores
eas submit --platform android
eas submit --platform ios
```
